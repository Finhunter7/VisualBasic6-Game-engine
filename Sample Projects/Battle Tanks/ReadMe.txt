This Is Sample/Exsample how to make a simple game using VBC engine version 0.2.0. This Project uses VBscript language.

How To Run This Project.

1. Execute VBCEPlayer.exe in the root of this Project. The Project file name must be Project.VBCEProject for this to work. 
This exe can be found on GitHub Relases page. 
You can also point the Project file to the VBCEPlayer.exe using command line arguments Ex: VBCEPlayer.exe C:\MyPath\MyProject.VBCEProject

2. Use The CodeEngine.Exe or VBCE Editor to open the .VBCEProject file. This can be also found on GitHub relases page
